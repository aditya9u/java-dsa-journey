
# Approach 1 Brute force

for this approach I need extra space and I can solve it with O(n)

Time Complexity --> O(n)
Space Complexity --> O(n)

# Approach 2 two pointer approach

for this approach I need 2 pointers where one will track index and 
other will go through the array and unique values updated in same array instead
of using extra space

Time Complexity --> O(n) // traversing an Array
Space Complexity --> O(1) // no extra space is needed
